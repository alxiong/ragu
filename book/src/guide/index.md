# User Guide

> Ragu is still under active development and documentation about its public API and recommended usage have not been written. [[tracking issue]](https://github.com/tachyon-zcash/ragu/issues/75)

## Security Warning

⚠️ **Ragu is under heavy development and has not undergone auditing.** Do not use this software in production.

## Dependencies and Requirements

* The minimum supported [Rust](https://rust-lang.org/) version is currently **1.85.0**.
* Ragu requires minimal dependencies and currently strives to avoid using dependencies that are not already used in [Zebra](https://github.com/ZcashFoundation/zebra).
* Ragu does not require usage of the standard library in code that is compiled for the target architecture.
